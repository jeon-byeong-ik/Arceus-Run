# 아르세우스 런 (Arceus Run)

2인용 터치 달리기 레이스 게임. 각자 화면 절반을 빠르게 **연타**해서 아르세우스를
달리게 하고, 아이템으로 가속·감속하며 먼저 결승선에 도달하면 승리합니다.

> **저작권 안내**: 모든 캐릭터 아트는 아르세우스에서 영감을 받은 **오리지널 팬 아트**(직접
> 그린 SVG 벡터)이며, 배경음악·효과음은 Web Audio API로 만든 **오리지널 칩튠**입니다.
> 공식 포켓몬 이미지나 음원은 사용하지 않은 비공식 팬 패러디 프로젝트입니다.

## 플레이 방법
- 📱 모바일: 화면 아래 파랑(1P)·빨강(2P) 패드를 연타
- ⌨️ 키보드: 1P = `F`, 2P = `J`
- 🍃 가속 · 🟣 감속 · 🛡️ 실드 · ⚡ 번개 부스트 아이템

## 특징
- 18종 타입 폼(노말~페어리) 캐릭터 선택, 타입별 색상 리컬러
- 6프레임 갤럽 애니메이션, 황금 십자 휠 + 초록 보석의 고퀄 SVG 스프라이트
- 패럴랙스 배경, 먼지/폭죽 파티클, 카운트다운, 진행 바, 포토피니시
- 오리지널 칩튠 BGM + 효과음, 음소거 지원
- 의존성 없는 **단일 HTML 파일**

## 실행
`index.html`(또는 `arceus-run.html`)을 브라우저에서 열면 바로 실행됩니다.

## 개발 방식 (SDD)
스펙 주도 개발. 전체 사양은 [`SPEC.md`](./SPEC.md) 참고.
